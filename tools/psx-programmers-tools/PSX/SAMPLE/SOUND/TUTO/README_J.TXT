<サンプルの解説>

  基本サウンドライブラリを使用したサンプルです。

tuto1:
======

<サンプルの解説>

  DMA 転送で送られた sin 波を、コントロールパッドの操作に
  よってさまざまな音程で発音するプログラムです。

<起動方法> 

  > psymake tuto1	コンパイル
  > run tuto1		実行

<コントローラの操作方法> 

	←:	低いピッチで sin 波を発生
	↑:	  ^
	→:       :
	↓:	  :
	□:	  :
	△:	  :
	○:	  v
	×:	高いピッチで sin 波を発生

	SELECT:	終了

tuto2:
======

<サンプルの解説>

  ミュートのサンプルプログラムです。実行すると

	発音します (実行されると自動的に発音します)
	    ↓
	START でミュート ON
	    ↓
	START でミュート OFF
	    ↓
	START で終了

    の処理を行います。
	
<起動方法> 

  > psymake tuto2	コンパイル
  > run tuto2		実行

<コントローラの操作方法> 

	START:	状態を遷移

tuto3:
======

<サンプルの解説>

  割り込みのサンプルプログラムです。

  ・DMA 転送で

	sin 波		(割り込み起動チェック音)
	ピアノの音	(割り込みをかける波形)

    の波形を送ります。

  ・割り込みアドレスはピアノの波形のちょうど中間に
    設定してあります。

  ・発音させるとしばらくして割り込みがかかり、その確認として
    sin 波の音が発音されます。

<起動方法> 

  > psymake tuto3	コンパイル
  > run tuto3		実行

<コントローラの操作方法> 

	START:		発音
	SELECT:		消音
	START + SELECT:	終了

tuto4:
======

<サンプルの解説>

  ノイズ音源のサンプルプログラムです。

<起動方法> 

  > psymake tuto4	コンパイル
  > run tuto4		実行

<コントローラの操作方法> 

	↑:	sin 波/音程 up
	↓:	sin 波/音程 down
	△:	ノイズ/音程 up
	×:	ノイズ/音程 down

tuto5:
======

<サンプルの解説>

  波形データの分割転送のサンプルプログラムです。

  サウンドバッファ内 2 つの領域に同じ波形データを 2048 バイトずつ分割
  して転送し、発音を行います。

<起動方法> 

  > psymake tuto5	コンパイル
  > run tuto5		実行

<コントローラの操作方法> 

	SELECT:	終了

tuto6:
======

<サンプルの解説>

  リバーブのサンプルプログラムです。

<起動方法> 

  > psymake tuto6	コンパイル
  > run tuto6		実行

  リバーブのモードを変えて、ピアノを発音することができます。

<コントローラの操作方法> 

	START:		ピアノ音発音
	SELECT:		ピアノ音消音
	↑↓:		リバーブモードの変更
	START + SELECT:	終了
