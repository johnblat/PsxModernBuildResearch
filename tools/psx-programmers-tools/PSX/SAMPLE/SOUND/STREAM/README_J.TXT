<サンプルの解説>

  基本サウンドライブラリに含まれる SPU ストリーミングライブラリを使用
  したサンプルです。

tuto1:
======

<サンプルの解説>

  7 チャンネル (ボイス数 14) の SPU ストリームをコントロールパッドの操
  作によって再生/停止を行います。SPU ストリーミングの状態も画面に表示
  します。

<起動方法>

  > psymake		コンパイル
  > psymake load	必要なデータを転送
  > run tuto1		実行

<コントローラの操作方法> 

	↑:	ボール数 増
	↓:	ボール数 減
	×:	SPU ストリーム全チャンネル同時開始
	○:	SPU ストリーム全チャンネル同時停止
      [ □:	確認サウンド停止 (#define USE_CONFIRM 時に有効) ]
	△:	終了

	START + ↑:	SPU ストリーム 0 チャンネル開始
		→:	SPU ストリーム 1 チャンネル開始
		↓:	SPU ストリーム 2 チャンネル開始
		←:	SPU ストリーム 3 チャンネル開始
		△:	SPU ストリーム 4 チャンネル開始
		○:	SPU ストリーム 5 チャンネル開始
		×:	SPU ストリーム 6 チャンネル開始

	SELECT + ↑:	SPU ストリーム 0 チャンネル停止
		 →:	SPU ストリーム 1 チャンネル停止
		 ↓:	SPU ストリーム 2 チャンネル停止
		 ←:	SPU ストリーム 3 チャンネル停止
		 △:	SPU ストリーム 4 チャンネル停止
		 ○:	SPU ストリーム 5 チャンネル停止
		 ×:	SPU ストリーム 6 チャンネル停止

tuto2:
======

<サンプルの解説>

  CD ストリーミングと SPU ストリーミングをミックスさせたサンプルです。
  CD ストリーミングのコードは sample\cd\movie\tuto0.c をベースにしてい
  ます。

  SPU ストリーミングは上記 tuto1.c とほぼ同じコードを使い、7 チャンネ
  ル (ボイス数 14) の SPU ストリームをコントロールパッドの操作によって
  再生/停止を行います。

<起動方法>

  > psymake		コンパイル
  > psymake load	必要なデータを転送
  > run tuto2		実行

<コントローラの操作方法> 

	×:		SPU ストリーム全チャンネル同時開始
	○:		SPU ストリーム全チャンネル同時停止
	△:		アニメーション・リスタート
      [ □:		確認サウンド停止 (#define USE_CONFIRM 時に有効) ]

	START + ↑:	SPU ストリーム 0 チャンネル開始
		→:	SPU ストリーム 1 チャンネル開始
		↓:	SPU ストリーム 2 チャンネル開始
		←:	SPU ストリーム 3 チャンネル開始
		△:	SPU ストリーム 4 チャンネル開始
		○:	SPU ストリーム 5 チャンネル開始
		×:	SPU ストリーム 6 チャンネル開始

	SELECT + ↑:	SPU ストリーム 0 チャンネル停止
		 →:	SPU ストリーム 1 チャンネル停止
		 ↓:	SPU ストリーム 2 チャンネル停止
		 ←:	SPU ストリーム 3 チャンネル停止
		 △:	SPU ストリーム 4 チャンネル停止
		 ○:	SPU ストリーム 5 チャンネル停止
		 ×:	SPU ストリーム 6 チャンネル停止

	START + SELECT:	終了
