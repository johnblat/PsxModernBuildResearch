cdvol: CD peak level meter

		     Copyright(C) 1997 Sony Computer Entertainment Inc.
						   All rights reserved.

<Descriptions>

  Display CD volume status by reading CD/DA data from SPU.  This is
  the sample for SpuReadDecodedData() and SpuSetTransferCallback().

  **** Notice ****
  This sample program works with CD/DA disc (anything you have)
  inserted in DTL-H2010 (or in DTL-H2510).

<Controller Button Assignment>

  Not assigned.

<How to execute>

  > psymake	compile
  > run main	run
