tuto0: 汎用的な HMD ビューワ

＜サンプルの解説＞
	HMD データフォーマットを使用した汎用的なビューワです。
	現在サポートされている全てのタイプのプリミティブドライバをサポート
	しています。

＜起動方法＞
	psymake tuto0.cpe	コンパイル
	psymake load0		必要なデータをダウンロード
	run tuto0		実行

＜コントローラの操作方法＞
	○,△,□,× 	： オブジェクトの回転
	→,←,↑,↓ 	： オブジェクトの XY 軸平行移動
        L1,R1	  	： オブジェクトの Z 軸平行移動
        L2,R2  		： 影響を受ける COORDINATE の変更

	START		： 影響を受ける COORDINATE を 0 番にする
	SELECT		： 影響を受ける COORDINATE を COORDINATE の数 -1 番
			   にする

	START & SELECT	： 終了

＜モデリングデータ＞
	tri.hmd		： トリケラトプス
	dino.hmd	： 恐竜（ティラノザウルス）
	cube.hmd	： 立方体
	tile.hmd	： 立方体（繰り返しテクスチャ）
	ground.hmd	： 地面


tuto1: 共有ポリゴンを使用した HMD ビューワ

＜サンプルの解説＞
	HMD データフォーマットの共有ポリゴンを使用したビューワです。
	共有ポリゴンとは、複数のマトリックスにまたがるポリゴンのことです。
	
	この共有ポリゴンを関節などに使用することにより、ワンスキンモデルが
	可能になります。

＜起動方法＞
	psymake tuto1.cpe	コンパイル
	psymake load1		必要なデータをダウンロード
	run tuto1		実行

＜コントローラの操作方法＞
	○,△,□,× 	： オブジェクト全体の回転
	→,←,↑,↓ 	： オブジェクト全体の XY 軸平行移動
        L1,R1 	 	： オブジェクト全体の Z 軸平行移動

	START を押しながら
	○,△,□,× 	： オブジェクト 1 の回転
	→,←,↑,↓ 	： オブジェクト 1 の XY 軸平行移動
        L1,R1	  	： オブジェクト 1 の Z 軸平行移動

	SELECT を押しながら
	○,△,□,× 	： オブジェクト 2 の回転
	→,←,↑,↓ 	： オブジェクト 2 の XY 軸平行移動
        L1,R1	  	： オブジェクト 2 の Z 軸平行移動

	START & SELECT	： 終了

＜モデリングデータ＞
	flag.hmd	： フランス国旗


tuto2: HMD を使用したシャトルビューワ

＜起動方法＞
	psymake tuto2.cpe	コンパイル
	psymake load2		必要なデータをダウンロード
	run tuto2		実行

＜コントローラの操作方法＞
	○△□× 	： オブジェクトの回転
	→←↑↓ 	： オブジェクトの XY 軸平行移動
        START SELECT  	： オブジェクトの Z 軸平行移動
        L１ R１  	： ハッチの開閉
        L２ R２  	： 衛星の送出

	START & SELECT	： 終了

＜モデリングデータ＞
	shuttle.hmd	： シャトル

＜サンプルの解説＞
	HMD データフォーマットの階層構造を使用したビューワです。

	データ作成時にあらかじめ親子関係やマトリックスをデータの中に埋め込む
	ことが出来るため、プログラムとデータを独立して扱うことが出来ます。

	このデータの中では、オブジェクトの使い回しを行っています。（ハッチや
	衛星の羽根など）


tuto3: 分割を使用した HMD ビューワ

＜サンプルの解説＞
	HMD データフォーマットの固定分割および自動分割を使用したビューワです。
	HMD では分割を行う場合は、プリミティブドライバを付け替えることで
	行います。さらに分割数を、データ内のプリミティブインデックスの上位 
	行います。さらに分割数を、データ内のプリミティブインデックスの上位 
	8 bit に指定します。

		+---------------------------------+
		|              TYPE               |
		+----------------+----------------+
		|   ポリゴン数   |     サイズ     |
		+-------+--------+----------------+
		|分割数 |        PRIM IDX         |
		+-------+--------+----------------+

	分割数を指定するマクロ（GsUNIT_DIV1〜DIV5）は libgs.h にあります。

	さらに、自動分割を行う場合は、GsSetAzwh() を使用して、Z 値、ポリゴン
	の大きさなどの分割の条件を設定する必要があります。

＜起動方法＞
	psymake tuto3.cpe	コンパイル
	psymake load3		必要なデータをダウンロード
	run tuto3		実行

＜コントローラの操作方法＞
	○,△,□,× 	： オブジェクトの回転
	→,←,↑,↓ 	： オブジェクトの XY 軸平行移動
        L1,R1  		： オブジェクトの Z 軸平行移動
        L2,R2	  	： 分割のモード切替え

	START		： ライン表示オン
	SELECT		： ライン表示オフ

	START & SELECT	： 終了

＜モデリングデータ＞
	jimen.hmd	： 地面


normview: HMD 法線ビューワ

＜サンプルの解説＞
	HMD データフォーマットを使用した法線ビューワです。
	フラットポリゴンの場合は、ポリゴンの中心から法線ベクトルの方向に
	赤いラインを表示します。また、グーローポリゴンの場合には、頂点ご
	とに表示します。

	表示は、../common にあるサンプルプリミティブドライバを使用します。
	現在、以下のサンプルプリミティブドライバが用意されています。

		GsU_f0000008:	フラット三角形
		GsU_f0000009:	フラットテクスチャ三角形
		GsU_f000000c:	グーロー三角形
		GsU_f000000d:	グーローテクスチャ三角形
		GsU_f0000010:	フラット四角形
		GsU_f0000011:	フラットテクスチャ四角形
		GsU_f0000014:	グーロー四角形
		GsU_f0000015:	グーローテクスチャ四角形
	

＜起動方法＞
	psymake normview.cpe	コンパイル
	psymake load0		必要なデータをダウンロード
	run normview		実行

＜コントローラの操作方法＞
	○,△,□,× 	： オブジェクトの回転
	→,←,↑,↓ 	： オブジェクトの XY 軸平行移動
        L1,R1	  	： オブジェクトの Z 軸平行移動
        L2,R2  		： 影響を受ける COORDINATE の変更

	START		： 影響を受ける COORDINATE を 0 番にする
	SELECT		： 影響を受ける COORDINATE を COORDINATE の数 -1 番
			   にする

	START & SELECT	： 終了
