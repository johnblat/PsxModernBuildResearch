＜サンプルの解説＞
	ポリゴン(三角形)からポリゴン(壁)への点光源(黄色い小三角形)
	によるドロップシャドーを行なう

＜起動方法＞
        >psymake   	コンパイル
        >run main       実行

＜コントローラの操作方法＞
        ○,△,□,×	：全体が回転する
        L1		：物体が動く
        L2		：全体が回転する
        R1		：光源が動く
        R2		：全体が回転する
        SELECT		：終了
