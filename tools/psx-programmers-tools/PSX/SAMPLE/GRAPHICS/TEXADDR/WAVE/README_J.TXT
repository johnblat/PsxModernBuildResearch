＜サンプルの解説＞

        テクスチャアドレスをリアルタイムに変更することで、
        疑似環境マップや、屈折のような効果を出すサンプルです。

        tuto1〜tuto3 は、チューニングのサンプルになっています。
        またこのチューニングは、パフォーマンスアナライザを利用して
        行われました。
        main は、３つの場面のオートデモが付いています。

        tuto1  プールの中の時計
               DMPSXを使用
        tuto2  プールの中の時計
               さらにCPU処理のチューン(スクラッチパッドをスタックに割り当て)
        tuto3  プールの中の時計
               さらにGPU処理のチューン(ＯＴを分割)
        main   無操作状態で２分間の３つの場面のオートデモ


＜起動方法＞

    tuto1、tuto2、tuto3

        > psymake tuto1.cpe  コンパイル
        > psymake tuto1      データロードと実行

    main

        > psymake main.cpe   コンパイル
        > psymake main       データロードと実行

＜コントローラの操作方法＞

    tuto1、tuto2、tuto3

      コントローラー１
        ○,△ ：水面の回転
        □,× ：水面の回転
        →,← ：水面の平行移動
        ↑,↓ ：水面の平行移動
        L1    ：屈折率を上げる
        L2    ：屈折率を下げる
        R1,R2 ：水面のZ軸平行移動
        START ：(未使用)
        SELECT：終了

      コントローラー２
        ○    ：波の動きを再開
        △    ：時計の拡大
        □    ：波の動きを停止
        ×    ：時計の縮小
        →,← ：時計の平行移動
        ↑,↓ ：時計の平行移動
        L1    ：反射率を上げる
        L2    ：反射率を下げる
        R1,R2 ：時計のZ軸平行移動
        START ：水底の場面切り替え
        SELECT：波のレンダリング方法切り替え

    main

      コントローラー１
        ○    ：波の動きを再開(オートデモに復帰)
        △    ：波の動きを再開(オートデモは停止)
        □    ：波のパターンの変更(３パターン)
        ×    ：波の動きを停止(オートデモの停止)
        →,← ：水面の回転
        ↑,↓ ：水面の回転
        L1    ：屈折率を上げる
        L2    ：屈折率を下げる
        R1    ：反射率を上げる
        R2    ：反射率を下げる
        START ：場面切り替え(３場面)
        SELECT：終了

      コントローラー２
        ○    ：(未使用)
        △    ：時計の拡大
        □    ：(未使用)
        ×    ：時計の縮小
        →,← ：時計の平行移動
        ↑,↓ ：時計の平行移動
        L1,L2 ：(未使用)
        R1,R2 ：時計のZ軸平行移動
        START ：(未使用)
        SELECT：(未使用)
