<サンプルの解説>
	
	- PlayStation のグラフィックスの基本機能のデモンストレーションを
	  行ないます。
	
	- 各プログラムはメニューから起動されます。詳細は menu.h,
	  main.c を参照下さい。
	
	- 検討用に各デモを個別に切り出して使用したい場合は、main.c 内
	  の記述をご参照下さい。


    balls
   ────────────────────────────────
	ボール状のスプライトを表示します。
	
	
    diffuse
   ────────────────────────────────
	10x10x10 個の正方形の板あるいは、10x10x10 個のボールパターンを
	透視変換して表示します。
	
    mat	
   ────────────────────────────────
	これは、３次元空間上に配置されたスプライトに、次々に異なるテク
	スチャを張り付けてアニメーションを行なうサンプルです。アニメー
	ションに使用されるロボットは Matchang(まっちゃん）と呼ばれてい
	て、プログラム名もここから由来しています。
	
	アニメーションに必要なテクスチャパターンはすべてフレームバッファ
	にあらかじめ転送しておく必要があります。ここでは、4 bit テクス
	チャを 4 ページ使用しています。

	ここでは、テクスチャ付き４角形ポリゴン(POLY_FT4)プリミティブを
	スプライトとして使用します。 このプリミティブは視点の位置に関
	わらず常にスクリーンに並行に配置されなくてはならないため、ここ
	では、スプライトのローカルスクリーンマトリクスを強制的に単位行
	列に置き換えています。
	
    rgb24	
   ────────────────────────────────
	フレームバッファを 24bit モードで表示するためのサンプルプログ
	ラムです。 
	24bit モードの場合 RGB それぞれに 8bit (256色）、合計で 3byte 
	が割り当てらるため、実際の表示ピクセルとフレームバッファの x 
	アドレスは一致しません。 
	また 24bit モードでの描画はできませんが、フレームバッファ間、
	フレームバッファ・メインメモリ間の転送は可能になります。

    morph	
   ────────────────────────────────
	メッシュ構造を利用して曲面に２次元画像をマッピングするサンプル
	です。PlayStation のテクスチャマッピング機能は１次変換 （線形
	変換）のみですが、 ポリゴンを細かいメッシュに分割してメッシュ
	上の各点（代表点）のみを正しく計算することでさまざまな曲面にマッ
	ピングを行なうことができます。こうしたポリゴン分割の考え方は透
	視変換の際に歪まないテクスチャマッピングを行なう場合にも応用で
	きます。 

<起動方法> 
	
	>psymake main
	>run main

<コントローラの操作方法>
	
	方向キー : カーソルの移動
        ○ :       決定
        SELECT :   終了

<備考>
	- カーソルを「DEMO」に合わせて○を押すとデモが起動されます。
	- カーソルを「HELP」に合わせて○を押すとデモ内のキー配置が表示
	  されます。各デモのキー配置は HELP を参照下さい。
