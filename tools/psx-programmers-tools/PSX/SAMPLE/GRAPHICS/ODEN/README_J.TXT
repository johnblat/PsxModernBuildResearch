<サンプルの解説>


このサンプルプログラムは、libgsを使って光源をリアルタイムに計算する
デモプログラムです。

パッドによる操作により ３光源の色及び明るさを RGB独立に設定でき また光源の
位置をリアルタイムに動かすことができます。

対象となるモデルは プリミィティブな球、円錐、立方体で テクスチャを有り無しも
パッドによる操作で設定できます。

モデリングデータのなかで 立方体はテクスチャのゆがみを押えるために自動分割
ポリゴンの機能を使っています。



<起動方法>

	> psymake
	> psymake load
	> run oden

＜コントローラの操作方法＞

<上面キーを押してない時>

        ○ :	オブジェクトの回転
	△ :	オブジェクトの回転
	□ :	オブジェクトの回転
	× :	オブジェクトの回転
        → :	右側のオブエジェクとにテクスチャを張る
        ← :	左側のオブエジェクとにテクスチャを張る
        ↑ :	真ん中のオブエジェクとにテクスチャを張る
        L1 :	押されていない
        L2 :	押されていない
	R1 :	押されていない
	R2：	押されていない
	START :		オブジェクトを近づける
        SELECT :	オブジェクトを遠ざける
	START&SELECT : プログラムの終了


＜上面左手前、奥、右奥のキーを押している時＞
        ○ :	光源色の緑成分を増す
	△ :	光源色の赤成分を増す
	□ :	光源色の緑成分を減らす
	× :	光源色の赤成分を減らす
	方向キー : 光源の方向を変える
        L1 :	光源2の調整を行う
        L2 :	光源1の調整を行う
	R1 :	光源3の調整を行う
	START :	 光源色の青成分を増す
        SELECT : 光源色の青成分を減らす


＜上面右手前のボタンを押している時＞
	△ :	光源のアンビエントを増す
	× :	光源のアンビエントを減らす
	R2：	押されている
