<サンプルの解説>
	
	- このディレクトリには、低レベルグラフィックライブラリ libgpu の
	  レベルの簡単なチュートリアルプログラムが入っています。
	
	- 各プログラムはメニューから起動されます。詳細は menu.h,
	  main.c を参照下さい。
	
	- 検討用に各デモを個別に切り出して使用したい場合は、main.c 内
	  の記述をご参照下さい。

	---------------------------------------------------------------------
	ファイル	|内容
	----------------+----------------------------------------------------
	tuto0		|スプライトを表示する。
	tuto1		|オーダリングテーブルを使用して表示優先順位を決定
	tuto2		|libgte 関数を使用た回転・拡大の簡単な例
	tuto3		|簡単な３次元物体 (cube) を表示する。
	tuto4		|tuto3 に光源計算機能を加える。
	tuto5		|tuto3 にテクスチャを加える
	tuto6		|tuto3 にフォグ効果を加える。
	tuto7		|ローカル座標系を用いて複数のオブジェクトを表示する。
	tuto8		|横方向のみの 1D スクロールＢＧ
	tuto9		|上下左右にスクロールするセルタイプＢＧ
	tuto10		|３次元セルタイプＢＧ（バードビュー）
	tuto11		|疑似モザイクを使用したセルタイプＢＧ
	tuto12		|疑似ラインスクロール
	tuto13		|マルチウィンドウ
	---------------------------------------------------------------------

	各プログラムは、共通に以下のソースファイルを使用します。
	
	---------------------------------------------------------------------
	ファイル	|内容
	----------------+----------------------------------------------------
	init.c		|グラフィック環境を初期化する
	cube.c		|いろいろなタイプの立方体を描画する
	lscr.c		|ラインスクロール関数ソースコード
	---------------------------------------------------------------------
	
	各プログラムは、以下のテクスチャパターンを使用します。

	---------------------------------------------------------------------
	ファイル	|内容
	----------------+----------------------------------------------------
	a)mat0tex.c	|4bit 64x64 スプライトパターン （ページ０）
	  mat1tex.c	|4bit 64x64 スプライトパターン （ページ１）
	  mat2tex.c	|4bit 64x64 スプライトパターン （ページ２）
	  mat3tex.c	|4bit 64x64 スプライトパターン （ページ３）
	----------------+----------------------------------------------------
	b)far0tex.c	|8bit 256x256 テクスチャパターン
	  far1tex.c	|8bit 256x256 テクスチャパターン		
	  near0tex.c	|8bit 256x256 テクスチャパターン	
	  near1tex.c	|8bit 256x256 テクスチャパターン
	  wintex.c	|8bit 256x256 テクスチャパターン
	----------------+----------------------------------------------------
	c)bgtex.c	|4bit 32x32 BG セルパターン
	----------------+----------------------------------------------------
	d)balltex.c	|4bit 16x16 ボールパターン CLUT 32 エントリ
	---------------------------------------------------------------------
	
	a),c) は、
	0x00-0x7f	4bit CLUT （先頭から32バイトのみ使用）
	0x80-		256x256 4bit テクスチャページ
	
	b) は、
	0x00-0x7f	8bit CLUT （先頭から32バイトのみ使用）
	0x80-		256x256 8bit テクスチャページ
	
	の形でテクスチャパターンが格納されています。

<起動方法> 
	
	>psymake main
	>run main

<コントローラの操作方法>
	
	方向キー : カーソルの移動
        ○ :       決定
        SELECT :   終了

<備考>
	- カーソルを「DEMO」に合わせて○を押すとデモが起動されます。
	- カーソルを「HELP」に合わせて○を押すとデモ内のキー配置が表示
	  されます。各デモのキー配置は HELP を参照下さい。
