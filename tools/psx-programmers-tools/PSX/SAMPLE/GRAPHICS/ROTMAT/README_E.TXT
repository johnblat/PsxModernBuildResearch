<Descriptions>
	This sample shows the visual difference between
	RotMatrix,RotMatrixYXZ,RotMatrixZYX and
	RotMatrix_gte,RotMatrixYXZ_gte,RotMatrixZYX_gte.

<How to execute>
        psymake        compile
        run dif_rmat   run

<Controller Button Assignment>
	Square Button           :The object rotates
        Cross Button            :The object rotates
        Triangle Button         :The object rotates
        Circle Button           :The object rotates
	L2,R2			:The object rotates
        L1,R1			:The object moves
        SELECT          	:Quit
