＜サンプルの解説＞
	RotMatrix,RotMatrixYXZ,RotMatrixZYXと
	RotMatrix_gte,RotMatrixYXZ_gte,RotMatrixZYX_gteを
	それぞれ比較する

＜起動方法＞
        >psymake        コンパイル
        >run dif_rmat       実行

＜コントローラの操作方法＞
        ○,△,□,×     ：物体ごとに回転する
	L2,R2		：物体ごとに回転する
        L1              ：物体が遠ざかる
        R1              ：物体が近づく
        SELECT          ：終了
