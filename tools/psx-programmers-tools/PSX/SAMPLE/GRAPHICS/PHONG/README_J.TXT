＜サンプルの解説＞
      このプログラムはPlayStationでフォンシェーディングを行なう
      サンプルです。PlayStationはフォンシェーディングを行なうハードウェア
      を持っていませんのでソフトで計算する必要があります。そのため
      グーローシェーディングにくらべて一桁以上計算時間がかかります。
      必要なポリゴン、効果的なポリゴンにだけフォンシェーディングをする
      ようにしてください。また、ポリゴンの数を増やせばグーローでも同じような
      効果は出せ、そのほうが高速です。
      このサンプルでは２つの８角柱のコーナーの面にだけフォンシェーディング
      を行なっています。スタート時の左がフォン,右がグーローです。
      コーナーの面積が小さいときはグーローに切り換えています。
      フォンシェーディングの結果は一旦、テクスチャ領域に描画され
      テクスチャーポリゴンとしてもう一度表示領域に描画されます。


＜起動方法＞
        >psymake   	コンパイル
        >run main       実行
