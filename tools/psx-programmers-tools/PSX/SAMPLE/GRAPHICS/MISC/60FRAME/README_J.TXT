60frame : 30フレーム/秒と60フレーム/秒の違いを表現したサンプル

<サンプルの解説>

        30フレーム/秒と60フレーム/秒の違いを見るためのサンプルです。
        切り替えて違いを比べてみてください。


<起動方法>

        >psymake        コンパイル
        >psymake load   必要なデータをダウンロード
        >run main       実行

<コントローラの操作方法>

        ○ : オブジェクトの回転（右）
        △ : オブジェクトの回転（上）
        □ : オブジェクトの回転（左）
        × : オブジェクトの回転（下）
        方向キー : オブジェクトの平行移動
        L1 : オブジェクトの平行移動（手前）
        L2 : オブジェクトの平行移動（奥）
        START  : 30フレーム/秒と60フレーム/秒の切り替え
        SELECT : プログラム終了
