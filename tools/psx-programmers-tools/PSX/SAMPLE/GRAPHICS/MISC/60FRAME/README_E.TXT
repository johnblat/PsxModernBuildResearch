60frame : demonstration of difference between 30 fps and 60 fps

                      Copyright(C) 1997 Sony Computer Entertainment Inc.
                                                    All Rights Reserved.

<Description>

        This sample shows difference between 30 fps and 60 fps.
        Please change frame rate and check the difference.

<How to execute>

        > psymake        compile
        > psymake load   load data
        > run main       run

<Controller Button Assignment>

        Up Directional Button    : Translate the obeject (UP)
        Down Directional Button  : Translate the obeject (DOWN)
        Right Directional Button : Translate the obeject (RIGHT)
        Left Directional Button  : Translate the obeject (LEFT)
        Square Button            : Rotate the object (LEFT)
        Cross Button             : Rotate the object (DOWN)
        Triangle Button          : Rotate the object (UP)
        Circle Button            : Rotate the object (RIGHT)
        L1 Button                : Translate the object Z axis (NEARER)
        L2 Button                : Translate the object Z axis (FARER)
        Start Button             : Switch between 30 fps and 60 fps
        Select Button            : Exit the program
