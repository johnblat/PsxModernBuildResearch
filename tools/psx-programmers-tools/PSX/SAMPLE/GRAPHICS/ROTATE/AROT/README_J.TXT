＜サンプルの解説＞
	このプログラムは任意の回転マトリクスが与えられたとき
	単位マトリクスとの間をある１本の回転軸のまわりの回転で
	内挿します。
	プログラムをスタートしたときの状態が単位マトリクスに対応しています。
	この状態からキューブをいろいろな向きに回転して任意の状態をつくります。
	この状態はある回転マトリクスで表現されます。
	ここで「内挿スタート」を押すと初期状態から最終状態への内挿が
	行なわれます。
	このサンプルはこれらの２つの状態を回転角の内挿やマトリクスの成分の
	内挿を使わず、回転軸を見つけることで行ないます。
	したがって、回転の方向が変化したり、大きさや形が変化して
	不自然な内挿になることがありません。

＜起動方法＞
	>psymake    	コンパイル
        >run main       実行

＜コントローラの操作方法＞
        ○	：Ｙ軸まわりの回転
        △	：Ｘ軸まわりの回転
        □	：Ｙ軸まわりの回転
        ×	：Ｘ軸まわりの回転
        →	：Ｘ軸方向の移動
        ←	：Ｘ軸方向の移動
        ↑	：Ｙ軸方向の移動
        ↓	：Ｙ軸方向の移動
        R1	：内挿スタート
        SELECT	：終了
