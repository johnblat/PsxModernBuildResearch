＜サンプルの解説＞

階層構造をもったモデルを、関節の座標に対する多重内挿法 (joint-MIMe) と頂点
の多重内挿法 (vertex-MIMe)によって操作するサンプルです。

また、joint-MIMe で用いる角度の内挿方法について、回転軸による内挿 
(axes-MIMe) および ロール・ピッチ・ヨー角による内挿 (RPY-MIMe) の２種類を
デモします。

tuto0 では、joint-MIMe と vertex-MIMe を同時に使用した多重内挿の例をしめします。

tuto1 では、joint-MIMe のみによる内挿の例をしめします。
	また、tuto1 ではモデルの状態を操作することができます。

＜起動方法＞ 

    - tuto0 
	> psymake all	コンパイル
	> psymake load0 必要なデータのロード
	> run tuto0	実行

    - tuto1
	> psymake all	コンパイル
	> psymake load1 必要なデータのロード
	> run tuto1	実行

＜コントローラの操作方法＞
・パッドボタンのわりあて
	パッド 1: プリセットモデルや、独自に変形したモデルを使って
		 MIMe 動作のようすを見るのに使います。
	パッド 2: 主にモデルの変形に使います。(tuto1 のみ)


	* パッド 1: 
		L1,L2,R1,R2:	MIMe パラメータ操作
			L1	MIMe 動作 1
			L2	MIMe 動作 2
			R1	MIMe 動作 3
			R2	MIMe 動作 4
			画面にそれぞれの MIMe 係数(重み)が表示されます

		←,→:	プリセットの選択 (tuto1 のみ)
		○,△,□,×:	視点の変更


		START:	角度内挿方法の変更
				axes-MIMe (回転軸の多重内挿) と
				RPY-MIMe (ロール・ピッチ・ヨー角の多重内挿)のトグル
				(画面に状態が表示されます)
		SELECT:	現在のモデルの状態をダンプする (tuto1 のみ)
			(preset.c へ追加するとプリセットとして利用できます)

	* パッド 2 (tuto1 のみ):
		R1,R2,○,△,□,×:
			回転 もしくは 移動

		START:	回転を操作するモードへ (画面に "ROTATION" と表示)
		SELECT:	移動を操作するモードへ (画面に "TRANSLATION" と表示)

		方向キー	操作対象の変更 (カーソルのある部分が操作対象)
			↑,↓:	操作モデルの変更 (MIMe1/MIMe2/MIMe3/MIMe4/ALL)
					base モデルにカーソルがあるときには
					すべてのモデルを同時に変形させます
			←,→:	操作オブジェクトの変更



＜備考＞

・このサンプルの使い方

1. まず起動したら、パッド 1 のL1,L2,R1,R2 ボタンをゆっくり押してみてください。
	多重内挿の様子が画面中央に表示されます。
	四つのボタンがそれぞれの MIMe のキーに対応しています。
	画面左のモデルが base, 画面右の4つが MIMe1-MIMe4です。

             +-----------------------------+
             |                             |
             |                   MIMe-key1 |
             |                             |
             |base  内挿された   MIMe-key2 |
             |         モデル              |
             |                   MIMe-key3 |
             |                             |
             |                   MIMe-key4 |
             +-----------------------------+

                    画面の様子


2. (tuto1 のみ) パッド 1 の方向キーの左右を押すと、次のプリセット状
	態へ移ります。現在のプリセット番号は画面左上２行目に表示されています。

3. パッド 1 の start ボタンを押すと、ロール・ピッチ・ヨー角による内挿と、
	回転軸による内挿の切替ができます。現在の内挿方法は画面左上１行
	目に表示されています。

4. (tuto1 のみ) パッド 2 を使うと、自分で好きな状態を作ることができます。
	方向キーで操作対象を変更し、○,△,□,×,R1,R2 ボタンで回
	転／移動をさせることができます。回転と移動のどちらを操作するの
	かは SELECT,START ボタンで変更します。現在の操作対象は画面上に
	表示されています。編集中にパッド 1 の方向キーの左右を押す
	と、プリセット状態へ復帰します。

・モデルの階層構造

	起動時に表示された状態で、以下のような階層構造を持っています。

		   親       子       孫
		  +-----+--+-----+--+-----+
		・|     |・|     |・|     |
		  +-----+--+-----+--+-----+
		↑       ↑       ↑
		親の原点 子の原点 孫の原点 


・プログラムの説明
	main.c: 一般の描画に必要な初期設定およびメインループ(OT の描画)
	tuto0.c: モデルの操作と OT への登録。(tuto0 用)
	tuto1.c: モデルの操作と OT への登録。(tuto1 用)
	a_intr.c: joint MIMe のための関数 (axes-MIMe 使用)
	rpy_intr.c: joint MIMe のための関数 (RPY-MIMe 使用)
	nmime.c: vertex mime のための関数 (tuto0 用)
	control.c コンボリューションによる平滑化のための関数
	ctlfc.c: コンボリューションで使うテーブル
	preset.c: モデルの位置のプリセットテーブル
