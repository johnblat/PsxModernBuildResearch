＜サンプルの解説＞

	階層構造をもったオブジェクトを、Joint-MIMe (関節マイム) によっ
	て操作するサンプルです。Joint-Axes-MIMe と Joint-RPY-MIMe の二
	種類の内挿法をデモします。


＜起動方法＞
        >psymake all   コンパイル
        >psymake load   必要なデータをダウンロード
        >run tuto0       実行


＜コントローラの操作方法＞

	パッド1: プリセットモデルや、独自に変形したモデルを使って
		 MIMe 動作のようすを見るのに使います。
	パッド2: モデルの変形に使います。


	パッド1:
		L1,L2,R1,R2: MIMe パラメータ操作
			L1: MIMe 動作 1
			L2: MIMe 動作 2
			R1: MIMe 動作 3
			R2: MIMe 動作 4
				(画面に MIMe パラメータ表示)

		→: 次のプリセット状態へ移る
		←: 前のプリセット状態へ移る

		START:	内挿方法の変更
			Joint-Axes-MIMe (回転軸内挿による関節マイム) と
			Joint-RPY-MIMe (RPY角による関節マイム)のトグル
				 (状態は画面に表示されています)
	
		SELECT:	現在のモデルの状態を標準出力へダンプする
			独自に変形したモデルをダンプし、preset.c へ追加すると
			プリセットとして利用できます。

	パッド2:
		R1,R2,○,△,□,×
			回転 もしくは 移動

		START	回転を操作するモードへ (画面に "ROTATION" と表示)
		SELECT	移動を操作するモードへ (画面に "TRANSLATION" と表示)

		方向キー	操作対象の変更 (明るい部分が操作対象)
			↑,↓	操作モデルの変更 (MIMe1/MIMe2/MIMe3/MIMe4/ALL)
			←,→	操作オブジェクト(胴、足、二の足、など)の変更

＜備考＞

* このサンプルの使い方

1. まず起動したら、パッド 1 をもち、L1/L2/R1/R2 キーをゆっくり押してみてく
	ださい。画面左のモデルが base, 画面右の4つが MIMe1-MIMe4 、画
	面中央が内挿されたモデルです。


             +-----------------------------+
             |                             |
             |                    MIMe1-key|
             |                             |
             |base  内挿された  MIMe2-key  |
             |         モデル              |
             |                    MIMe3-key|
             |                             |
             |                  MIMe4-key  |
             +-----------------------------+

                    画面の様子


2. パッド 1 の←,→を押すと、次のプリセット状態へ移ります。

3. パッド 1 の START ボタンを押すと、ロール・ピッチ・ヨー角による内挿と、
	回転軸による内挿の切替ができます。現在の内挿方法は画面左上２行目
	に表示されています。

4. パッド 2 を使うと、自分で好きな状態を作ることができます。

	←,→,↑,↓で操作対象を変更し、○,△,□,×,R1,R2 で回転・移動をさ
	せることができます。回転と移動のどちらを操作するのかは
	SELECT,START で変更します。現在の操作対象は画面上に表示されて
	います。編集中にパッド 1 の←,→を押すと、プリセット状態へ復帰
	します。


* モデルの階層構造

モデルの原点の位置を () に示します。

    WORLD
     |
     +- 胴 (腰)
         |
         +- 頭 (首)
         |
         +- 左もも (足の付け根)
         |   |
         |   +- 左ふくらはぎ (ひざ)
         |
         +- 右もも (足の付け根)
         |   |
         |   +- 右ふくらはぎ (ひざ)
         |
         +- 左上腕 (肩)
         |   |
         |   +- 左前腕 (ひじ)
         |
         +- 右上腕 (肩)
             |
             +- 右前腕 (ひじ)
