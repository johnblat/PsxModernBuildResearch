<サンプルの解説>
	
	- ここには、高レベルライブラリ libgs と低レベルライブラリ libgte 
	  を同時に使用したサンプルが収められています。いずれのサンプルも
	  libgs で処理された３Ｄオブジェクトと libgte で左上端点が計算さ
	  れた複数個のスプライトが同時に表示されます。
	
	- 各プログラムはメニューから起動されます。詳細は menu.h,
	  main.c を参照下さい。
	
	- 検討用に各デモを個別に切り出して使用したい場合は、main.c 内
	  の記述をご参照下さい。

	============================================================
	プログラム	内容				
	------------------------------------------------------------
 	tuto0		libgs の管理するＯＴに低レベルプリミティブを 
			AddPrim() 関数を使用して接続して表示する
	tuto1		低レベルのＯＴに libgs の管理する DOBJ オブジェ
			クトを接続して表示する
	------------------------------------------------------------

<起動方法> 
	
	>psymake main
	>psymake load
	>run main

<コントローラの操作方法>
	
	方向キー : カーソルの移動
        ○ :       決定
        SELECT :   終了

<備考>
	- カーソルを「DEMO」に合わせて○を押すとデモが起動されます。
	- カーソルを「HELP」に合わせて○を押すとデモ内のキー配置が表示
	  されます。各デモのキー配置は HELP を参照下さい。
