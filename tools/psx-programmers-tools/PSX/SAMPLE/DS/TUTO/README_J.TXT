tuto0 - tuto11:
===============

＜サンプルの解説＞

	これらは、libcd のチュートリアルを libds.lib 用に書き直した
	ものです。
	プログラムをできるだけ同じ構成にしてあるので、CD のアクセス
	方法が libds.lib に最適な方法とは限りませんが、libcd のチュ
	ートリアルと比較することで、libds の特徴がつかめると思います。

tuto12:
=======

＜サンプルの解説＞

	先行シークで DA を再生した場合の曲のスタートまでの時間を、そ
	うでない場合と比較したサンプルです。

	画面に表示される "TIME TO START" が再生が始まるまでの時間（
	フレーム数：1/60 sec 単位）です。これにより、シークにかかる
	時間を把握することができます。

＜起動方法＞ 

	> psymake tuto12
	> run tuto12

＜コントローラの操作方法＞

	方向キー：	曲の選択（先行シーク時は曲の先頭にシークされる）
	△：		先行シーク ON/OFF 切り替え
	×：		再生開始
	SELECT：	終了

＜備考＞

	実際のシークに要する時間は状態（ヘッドの位置、ファイルの位置、
	その他の環境）に左右されるため、このサンプルで得られるデータ
	は目安としてご使用下さい。

tuto14:
=======

＜サンプルの解説＞

	複数のファイルを連続でリードする場合（CD 上に連続的に記録し
	ていちいちシークせずにリードする）と、一つ一つシークしてリー
	ドした場合の、要する時間を比較するサンプルです。
	サンプルではデータセクタを５０セクタごとに区切って、一つ一つ
	をファイルと見なしてアクセスしています。

	画面に表示される "TIME TO READ" が５０セクタ * １０個のリー
	ドにかかった時間(フレーム数：1/60 sec 単位）です。

＜起動方法＞ 

	> psymake tuto14
	> run tuto14

＜コントローラの操作方法＞

	△：		連続リード／個別アクセスリード切り替え
	×：		リード開始
	SELECT：	終了

＜備考＞

	実際のシークに要する時間は状態(ヘッドの位置、ファイルの位置、
	その他の環境）に左右されるため、このサンプルで得られるデータ
	は目安としてご使用下さい。
