＜解説＞

浮動小数点演算、数学関数を使って樹木曲線を描く。
パッドの上下キーで曲率を指定し、○or×ボタンで開始。


＜起動方法＞ 

  >psymake
  >run main


＜コントローラの操作方法＞

	○または×：描画開始
	↑または↓：曲率の指定
