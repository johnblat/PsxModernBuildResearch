<Description>

  This program draws tree curve by floating-point calculation and 
  mathmatic function.


<How to Execute>

  >psymake
  >run main


<Controller Button Assignment>

        Circle or Cross Button        : start drawing
        Up or Down Directional Button : setting of curve ratio
