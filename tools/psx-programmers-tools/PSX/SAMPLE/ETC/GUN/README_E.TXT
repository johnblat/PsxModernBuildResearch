<Description>
	This program uses the libgun function to detect the position of the TV 
	screen towards which the gun controller is pointing and displays it on 
	screen.

<How to Execute>

	psymake
	run main

<Controller Button Assignment>

	Trigger button:	displays the position in which the gun is pointing
	Select button:	terminates the program.
