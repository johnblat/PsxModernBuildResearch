＜サンプルの解説＞
	libgunの関数を使用して、ガンコントローラが向いているＴＶ画面の
	位置を検出して、画面上に表示するプログラムです。

＜起動方法＞
	>psymake
	>run main


＜コントローラの操作方法＞
	トリガー	：ガンの向いている位置を表示
	SELECT		：終了
