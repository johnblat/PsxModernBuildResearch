＜サンプルの解説＞
	デバッグステーションとＰＣをH3050で接続してＰＣからの入力を
	エコーバックするプログラムです。標準入出力のドライバをもち
	いずに、BIOSを用いて、受信割り込みによるコールバックを実現
	しています。

        >psymake
        >run tuto2
