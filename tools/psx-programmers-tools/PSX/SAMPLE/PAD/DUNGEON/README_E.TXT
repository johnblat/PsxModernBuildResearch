<Description>

	This sample contents contains moves within the dungeon  by using
        the Analog stick. The motion speed and shock is expressed by 
        by the Actuator.

<How to execut >

	>psymake	compile
	>psymake load	data transmission
	>run main	execute

<Controller button assignment>

	Analog Button     :  change of controller mode
	Left stick        :  move
	Right stick       :  not used
