snowman: ライトプリミティブを使用したサンプルデータ

＜サンプルの解説＞
	当データは、HMD の装置プリミティブを使用した、ライトを設定するデモン
	ストレーションです。
	1 つの HMD データの中で、3 種類の異なるタイプのライトを使用していま
	す。

	HMD サンプルプログラムディレクトリの anim/animview を用いて
	再生することができます。

	このデータは、以下の手順で作成されました。

	1. LightWave 3D でモデルデータとアニメーションが作成されました。
	2. 株式会社ディ・ストーム様により、開発/提供中の "Animation Saver for
           PlayStation" version 1.1.2 により、LAB ファイルが生成されました。
	3. テキストエディタを用いて、装置プリミティブタイプが編集されました。
	4. 以上の結果から、HMD アセンブラ (labp) により HMD ファイル
	   "snowman.hmd" が生成されました。
