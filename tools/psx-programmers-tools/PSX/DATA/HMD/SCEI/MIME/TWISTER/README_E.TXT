twister: combination of Joint and Vertices MIMe

<description>
	This data shows a combination of Joint and Vertices MIMe at same time.
	Please use mime/tuto1 in HMD sample program directory to run.

	This data is created by the following steps:

	1. "gen" is compiled from gen.c.
	2. base.hmd, bend.hmd, twist.hmd and popye.hmd are generated by
	   "gen" and "labp".
	3. "mkmime" is used to generate axes.hmd and rpy.hmd.
