twister: Joint MIMe と頂点 MIMe の組合せデモ

＜サンプルの解説＞
	このサンプルは、Joint MIMe と頂点 MIMe を同時に行うデモです。
	HMD サンプルプログラムディレクトリの mime/tuto1 を用いて
	再生することができます。

	このデータは、以下の手順で作成されました。

	1. gen.c から、"gen" が C コンパイラで生成されました。
	2. "gen" と "labp" により、base.hmd, bend.hmd, twist.hmd, popye.hmd が
	   生成されました。
	3. "mkmime" コマンドにより、axes.hmd と rpy.hmd が生成されました。
