puyo: 頂点/法線 MIMe

＜サンプルの解説＞
	このサンプルデータは、頂点/法線 MIMe のデモです。
	HMD サンプルプログラムディレクトリの mime/tuto0 を用いて
	再生することができます。

	当サンプルデータは、TMD データを元にバイナリエディタと、
	"xhmd", "labp" コマンドで作成されました。
