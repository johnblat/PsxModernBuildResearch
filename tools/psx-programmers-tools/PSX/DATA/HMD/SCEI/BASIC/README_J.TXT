cube: キューブ群

＜サンプルの解説＞
	このサンプルデータは、キューブ群を表示します。
	HMD サンプルプログラムディレクトリの basic/tuto0 を用いて
	再生することができます。

	当サンプルデータは、TMD/TIM データを元にバイナリエディタで
	作成されました。

shuttle: 階層構造デモ

＜サンプルの解説＞
	このデータは、階層構造を持つスペースシャトルを表示します。
	このスペースシャトルは TMD でのデモでも知られていますが、
	TMD 版では、階層構造は C コードに記述されていました。
	この HMD 版デモでは、全ての階層情報はデータファイルだけに記述
	されています。
	HMD サンプルプログラムディレクトリの basic/tuto2 を用いて
	再生することができます。

	当サンプルデータは、TMD データを元にバイナリエディタで
	作成されました。

tri: ワンスキンモデルのトリケラトプス

＜サンプルの解説＞
	このサンプルデータは、ワンスキンモデルのデモです。
	HMD サンプルプログラムディレクトリの basic/tuto0 を用いて
	再生することができます。

	当サンプルデータは、以下の手順で作成されました。

	1. 旧 RSD フォーマットファイルから、新 RSD ファイルを作成するために、
	   "rsdup" と "meshconv" が用いられました。
	2. 上記で作成された RSD ファイルと TIM ファイルから "rsd2hmd" を用いて
	   tri.hmd が作成されました。
