dice: 繰り返しテクスチャ

＜サンプルの解説＞
	繰り返しテクスチャ指定パラメータの使い方を示すデモです。
	HMD サンプルプログラムディレクトリの basic/tuto0 を用いて
	再生することができます。

	このデータは、以下の手順で作成されました。

	1. LightWave 3D 用 HMD プラグイン "PS AnimationSaver" を用いて、
	   LAB ファイルが生成されました。
	2. 1 で生成されたファイルをテキストエディタで編集し、TEX_WIN,
	   CBA, TSB 等のマクロを用い、より意味がわかりやすく変更されました。
	3. "labp" コマンドにより、2 で作られたファイルから、最終的な HMD
	   ファイルが生成されました。
