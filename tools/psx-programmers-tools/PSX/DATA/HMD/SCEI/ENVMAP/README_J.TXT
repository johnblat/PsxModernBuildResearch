star09: 星郡

＜サンプルの解説＞
        このサンプルデータは、環境マップ星群を表示します。
        HMD サンプルプログラムディレクトリの basic/tuto0 を用いて
        再生することができます。
        anim/animview を用いるとアニメーションが再生できます。

        当サンプルデータは、LightWave 3D を用いて作成されたデータをもとに、
        "xhmd", "labp" コマンドを使用して作成されました。
