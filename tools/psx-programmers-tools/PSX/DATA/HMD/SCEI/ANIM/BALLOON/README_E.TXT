balloon: a balloon with bezier curve interpolated animation

<description>
	This data shows a sample of non-linear interpolated animation,
	in this case, bezier curve.
	Please use anim/animview in HMD sample program directory to run.

	This data is created by the following steps:

	1. Base model is created by LightWave 3D and converted to balloon.lab.
	2. balloon.lab is edited by text editor to add shared polygons as
	   the straps, and animation for the cabin.
	3. "labp" command created balloon.hmd from balloon.lab.
