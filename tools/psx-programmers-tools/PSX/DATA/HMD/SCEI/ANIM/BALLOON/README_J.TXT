balloon: Bezier 曲線補間によるアニメーション

＜サンプルの解説＞
	当サンプルデータは、非線形補間アニメーションのデモです。
	このデータでは、Bezier 曲線補間を用いています。
	HMD サンプルプログラムディレクトリの anim/animview を用いて
	再生することができます。

	このデータは、以下の手順で作成されました。

	1. 基本形が LightWave 3D により作成され、balloon.lab に変換されました。
	2. テキストエディタにより balloon.lab を編集し、吊り紐のための
	   共有ポリゴンと、アニメーションが付加されました。
	3. "labp" コマンドを用い、balloon.lab から balloon.hmd が
	   生成されました。
