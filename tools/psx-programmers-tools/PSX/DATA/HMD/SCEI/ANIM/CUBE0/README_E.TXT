cube0: animated cubes

<description>
	This sample data shows a set of cubes that is animated by HMD animation
	primitive. Please use anim/tuto0 in HMD sample program directory to run.

	This data is created by the following steps:

	1. "rsdup" is used to create new-RSD format files (cube0.rsd,
	   cube0.ply, cube0.mat and cube0.grp) from old-RSD.
	2. cube0.cod and cube0.mot are created by text editor.
	3. "rsd2hmd" created cube0.hmd from the above created files and
	   TIM files.
