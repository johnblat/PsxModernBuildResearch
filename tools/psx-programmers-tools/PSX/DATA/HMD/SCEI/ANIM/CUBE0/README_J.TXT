cube0: アニメーションキューブ

＜サンプルの解説＞
	このサンプルデータは、HMD アニメーションを行うキューブの集合です。
	HMD サンプルプログラムディレクトリの anim/tuto0 を用いて
	再生することができます。

	当サンプルデータは、以下の手順で作成されました。

	1. 旧 RSD フォーマットファイルから、新 RSD ファイル (cube0.rsd,
	   cube0.ply, cube0.mat, cube0.grp) を作成するために、"rsdup"
	   が用いられました。
	2. テキストエディタにより、cube0.cod と cube0.mot が作成されました。
	3. 最後に、上記で作成された RSD ファイルと TIM ファイルから
	   "rsd2hmd" を用いて cube0.hmd が作成されました。
