snake: 蛇のような動き

＜サンプルの解説＞
	このサンプルは、シーケンス開始インデックスのずれによるアニメーション
	を行うデモです。
	HMD サンプルプログラムディレクトリの anim/animview を用いて
	再生することができます。

	このデータは、以下の手順で作成されました。

	1. gen.c から、"gen" が C コンパイラで生成されました。
	2. "gen" により 04.lab と 64.lab が生成されました。
	3. "labp" により、04.hmd と 64.hmd が生成されました。
	4. gen2.c から、"gen2" が C コンパイラで生成されました。
	5. "gen2" により 04_2.lab と 64_2.lab が生成されました。
	6. "labp" により、04_2.hmd と 64_2.hmd が生成されました。
