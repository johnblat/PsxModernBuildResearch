snake: behavior like a snake

<description>
	This data shows an animation that is animated by difference of starting
	sequence index.
	Please use anim/animview in HMD sample program directory to run.

	The following steps create this data:

	1. "gen" is compiled from gen.c.
	2. 04.lab and 64.lab are generated by "gen".
	3. 04.hmd and 64.hmd are generated by "labp".
	4. "gen2" is compiled from gen2.c.
	5. 04_2.lab and 64_2.lab are generated by "gen2".
	6. 04_2.hmd and 64_2.hmd are generated by "labp".
