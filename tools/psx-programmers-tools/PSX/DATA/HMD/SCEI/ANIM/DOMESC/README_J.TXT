	カラードポリゴンで構成されています。../domes/readme_j.txt も
	併せてご参照ください。

	あります。HMD サンプルプログラムディレクトリの anim/animview を用いて
	再生してください。

	このデータは、以下の手順で作成されました。

	1. 以下のコマンドラインが、カラードポリゴンの生成と、各頂点に
	   異なる色を付加するために、IRIX 上で実行されました。

		tr -d '^M' <../domes/domes.lab |
			./iipcol.sh |./mark.sh |./col.sh >domesc.lab

	   最初のコマンド 'tr' は、../domes/domes.lab に含まれる DOS 改行を
	   UNIX のものに変換します。次に './iipcol.sh' により、光源計算無し
	   グーローポリゴンの各頂点に、異なる色が割り当てられました。
	   そして './mark.sh' により、各ポリゴンデータを示すラベルに
	   実際のポリゴンタイプをコメントとして追加しました。これは、
	   次の段階のための準備です。最後に './col.sh' を用いて、カラード
	   ポリゴンが生成されました。

	2. HMD アセンブラ (labp) により HMD ファイル "domes.hmd" が
	   生成されました。
