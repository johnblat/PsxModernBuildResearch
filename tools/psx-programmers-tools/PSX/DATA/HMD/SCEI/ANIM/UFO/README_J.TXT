ufo: UFO と足の骨

＜サンプルの解説＞
	Bezier 曲線補完によるキーフレームアニメーションのデモです。
	HMD サンプルプログラムディレクトリの anim/animview を用いて
	再生することができます。

	このデータは、以下の手順で作成されました。

	1. LightWave 3D のサンプルデータの "footbone.lwo" と "adamskiu.lwo" を
	   組み合わせたシーンから、HMD ファイルが作成されました。
	2. "xhmd" により、1 で生成された HMD ファイルが逆アセンブルされ、
	   編集可能ファイルが生成されました。
	3. 2 で生成されたファイルをテキストエディタで編集し、アニメーション
	   プリミティブデータの追加が行われました。
	4. "labp" コマンドにより、3 で作られたファイルから、最終的な HMD
	   ファイルが生成されました。
