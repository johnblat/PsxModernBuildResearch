ufo: UFO and foot born

<description>
	This data shows animation with bezier curve interpolation.
	Please use anim/animview in HMD sample program directory to run.

	The following steps create this data:

	1. An intermediate HMD file is generated based on LightWave 3D's sample
	   data "footborn.lwo" and "adamskiu.lwo".
	2. "xhmd" is used to dis-assemble the HMD file created in step 1.
	3. Text editor to add animation edits the result of step 2
	   primitive data.
	4. "labp" command created HMD files from *.lab files created in step 3.
