HMD プリミティブドライバソース公開について

Runtime Library Release 4.4 より、HMD プリミティブドライバのソースコードを
公開いたします。

目的:
	- ゲーム制作時におけるチューニングの参考にしていただくため。
	- HMD アーキテクチャを充分にご活用いただくため。
	- HMD プリミティブドライバをユーザ様で独自に作成していただく際の
	  ご参考として。

方法:
	- アセンブラによるものを含む、全てのプリミティブドライバソースコードを
	  公開します。
	- アセンブラソースは、ccpsx を用いてアセンブルすることができます。
	  マクロアセンブラ asmpsx ではアセンブルできませんのでご注意ください。
	- 各ソースコードは、PlayStation library の libhmd を作成するために
	  SCEI で用いられているものと完全に同じものです。
	- プリミティブドライバ以外のフレームワーク関数は公開いたしません。
	- ソースコードは、\psx\src\lib\hmd\ ディレクトリに格納されています。

備考:
	- 全ての著作権は、株式会社ソニー・コンピュータエンタテインメントに
	  帰属します。
	- ゲーム制作時において、提供されたソースコードを改変して組み込むことは
	  自由です。
	- ソースコードに含まれるバグ・不具合などに対する対応は、通常の
	  ライブラリに対するものと同等です。
