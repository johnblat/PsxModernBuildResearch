PlayStation COFF Library

This directory contains COFF format libraries which can be used with 
CodeWarrior, NEWS, or SGI.

When you use those libraries with CodeWarrior, please follow instruction of 
CodeWarrior for conversion.
 
DTL-H5000 user can use a copy of coff/lib, coff/include. 

These libraries can be used with following compilers.
 CodeWarrior for PS
 NEWS-OS 6.x MIPS-CC
 IRIX 5.3 MIPS-CC
 GNU-C generated for ECOFF target
