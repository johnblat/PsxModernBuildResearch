PlayStation COFFライブラリ

このディレクトリには CodeWarrior や NEWS、SGI で使用可能な coff形式の
ライブラリが入っています。
CodeWarrior で使用する場合は CodeWarrior の説明書に従って変換してください。

DTL-H5000 を使用している方は、coff/lib, coff/include をコピーして使用
してください。

このライブラリは以下のコンパイラで使用可能です。
 CodeWarrior for PS
 NEWS-OS 6.x の MIPS-CC
 IRIX 5.3 の MIPS-CC
 ECOFF をターゲットとして生成された GNU-C
