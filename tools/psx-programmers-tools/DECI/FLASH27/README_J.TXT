		DTL-H2700 DECI Firmware Kit


ファームウェア書き換え

  DTL-H2700 のフラッシュメモリに、ROMイメージを書き込みます。

    書式： flash <I/Oアドレス>

    <I/Oアドレス> は、DTL-H2700で設定してあるI/Oアドレスを指定します。

    例
        DOS> cd \deci\flash27
        DOS> flash 1340

    注意

    *正常に動作せず、DOSプロンプトに戻ってこない場合は F10 F2 を押し
    て下さい。
