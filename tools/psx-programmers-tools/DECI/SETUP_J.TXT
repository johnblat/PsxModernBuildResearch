 PlayStation DECI driver Version 1.4


 PlayStation DECIドライバのインストール

 ◆ はじめに
   PlayStation DECIドライバは、PlayStationの開発ボードを各種ツールより
  使用可能にする Windows(95/NT)用ドライバです。
   CodeWarrior R4 以降、もしくは DECIドライバが必要だと明記されたツール
  を使用する場合に下記手順においてインストールを行ってください。
   なお、dexbios, h25bios などで動作する SDevTC のツールを同時使用でき
  なくなりますのでご注意ください。


 ◆ ＯＳ，機種
   以下の４つの組合せに分類してそれぞれ説明してありますので、お手元の
  ＯＳ，機種にしたがって最適なものを選択してお読みください。

  A. Ｗｉｎｄｏｗｓ９５ と DTL-H2000/DTL-H2700 の組合せ
  B. Ｗｉｎｄｏｗｓ９５ と DTL-H2500 の組合せ
  C. ＷｉｎｄｏｗｓＮＴ と DTL-H2000/DTL-H2700 の組合せ
  D. ＷｉｎｄｏｗｓＮＴ と DTL-H2500 の組合せ


 ◆ インストール

  A. Ｗｉｎｄｏｗｓ９５ に DTL-H2000/DTL-H2700 を接続する。

   A-1. ハードウェアのセットアップ

     開発ボードの取り扱い説明書、もしくはスタートアップマニュアルを
    参考にしてＰＣにDTL-H2000/DTL-H2700を接続してください。

   A-2. ボードの初期化 (DTL-H2700のみ)

     DTL-H2700は、ファームウェアを最新のバージョンにupdateしてください。
    DOSプロンプトもしくはDOSより以下のコマンドを実行することによって
    updateできます。DTL-H2000の場合はこの操作は必要ありません。
       > cd deci\flash27
       > flash 1340
       (1340はDTL-H2700のI/OAdressです。)

   A-3. CW R3以前の環境、SDevTC環境の設定解除

     CW R3以前のドライバが組み込まれていた場合には以前のドライバーを
    アンインストールしてください。
       スタートメニュー -> 設定 -> コントロールパネル
       システム -> デバイスマネージャ -> PlayStation -> DTL-H2000
       削除を選択してください。

     SDevTC環境 で dexbios の組み込みを config.sys などで実行している
    場合は、該当部分を config.sys などから外すかコメントにしてください。

   A-4. I/O Adress, IRQ の指定
     deci\psx95isa をハードディスクの適当なところへコピーしてください。
     コピーした deci\psx95isa\psx95isa.infをダブルクリックする。

     ;IRQConfig=で始まっている行を探してください。

     IRQを使用する場合は IRQConfig=10 のように最初の ;を削除した後 10,11,
    12,15 などのボードに設定した値を書き込んでください。
     IRQを使用しない場合は ;をつけたままにしておいてください。

     その直後に IOConfig=で始まっている行がみつかると思います。こちらに
    I/O アドレスを指定してください。IOConfig=1240-124b(3ff::)などのように
    してください。

     初期値は IRQ:なし、I/O アドレス:1340となってます。

   A-5. ドライバーのインストール
       スタートメニュー -> 設定 -> コントロールパネル
       ハードウェアをダブルクリックしてください。
       インストールを始めるで "次へ"を選択してください。
       自動的に検出で "いいえ"を選択して "次へ"を選択してください。
       "その他のデバイス"を選択して "次へ"を選択してください。
       "ディスク使用"を選択してください。
       配布ファイルのコピー元で 先程(A-4.)の設定をした psx95isa.infのある
       ディレクトリを指定してください。
       PlayStation Board DECI (DTL-H2000/H2700)で "次へ"を選択してください。
       ポートアドレスが表示されますので確認して "次へ"を選択してください。
       完了をおして、ドライバーのインストールは終了です。


  B. Ｗｉｎｄｏｗｓ９５ に DTL-H2500 を接続する。

   B-1. ハードウェアのセットアップ

     開発ボードの取り扱い説明書、もしくはスタートアップマニュアルを
    参考にしてＰＣにDTL-H2500を接続してください。

   B-2. ボードの初期化

     DTL-H2500は、ファームウェアを最新のバージョンにupdateしてください。
    DOSプロンプトもしくはDOSより以下のコマンドを実行することによって
    updateできます。
       > cd deci\flash25
       > flash

   B-3. CW R3以前の環境、SDevTC環境の設定解除

     ドライバを使用していなかった場合はその他のデバイスに PCI Card
    と認識されてますので、削除してください。
       スタートメニュー -> 設定 -> コントロールパネル
       システム -> デバイスマネージャ -> その他のデバイス -> PCI Card
       削除を選択してください。

     CW R3以前のドライバが組み込まれていた場合には以前のドライバーを
    アンインストールしてください。
       スタートメニュー -> 設定 -> コントロールパネル
       システム -> デバイスマネージャ -> PlayStation -> DTL-H2500
       削除を選択してください。

   B-4. ドライバーのインストール
       スタートメニュー -> 設定 -> コントロールパネル
       ハードウェアをダブルクリックしてください。
       インストールを始めるで "次へ"を選択してください。
       自動的に検出で "いいえ"を選択して "次へ"を選択してください。
       "その他のデバイス"を選択して "次へ"を選択してください。
       "ディスク使用"を選択してください。
       配布ファイルのコピー元で deci\psx95pci ディレクトリを指定してください。
       PlayStation Board DECI (DTL-H2500)で "次へ"を選択してください。
       プラグアンドプレイに対応していますで "次へ"を選択してください。
       完了をおして、ドライバーのインストールは終了です。


  C. ＷｉｎｄｏｗｓＮＴ に DTL-H2000/DTL-H2700 を接続する。

   C-1. ハードウェアのセットアップ

     開発ボードの取り扱い説明書、もしくはスタートアップマニュアルを
    参考にしてＰＣにDTL-H2000/DTL-H2700を接続してください。

   C-2. ボードの初期化 (DTL-H2700のみ)

     DTL-H2700は、ファームウェアを最新のバージョンにupdateしてください。
    起動可能なフロッピーディスクに deci\flash27以下をコピーし以下のコマ
    ンドを実行することによって updateできます。DTL-H2000の場合はこの操作
    は必要ありません。
       > cd deci\flash27
       > flash 1340
       (1340はDTL-H2700のI/OAdressです。)

   C-3. I/O Adress, IRQ の指定
     deci\psxNTisa をハードディスクの適当なところへコピーしてください。
     コピーした deci\psxNTisa\psxNTisa.infをダブルクリックする。

     ISAIRQ=で始まっている行を探してください。

     IRQを使用する場合は ISAIRQ=10 のように 10,11,12,15 などのボードに
    設定した値を書き込んでください。IRQを使用しない場合は 0をしてして
    ください。

     その直後に ISAAddress=で始まっている行がみつかると思います。こちらに
    I/O アドレスを指定してください。ISAAddress=0x1240などのようにしてください。

     初期値は IRQ:なし、I/O アドレス:1340となってます。

   C-4. ドライバーのインストール
     先程(C-3.)の設定をした psxNTisa.infを右クリックしてください。
     メニューの中にインストールがあるので選択してください。
     ドライバーのインストールは終了です。


  D. ＷｉｎｄｏｗｓＮＴ に DTL-H2500 を接続する。

   D-1. ハードウェアのセットアップ

     開発ボードの取り扱い説明書、もしくはスタートアップマニュアルを
    参考にしてＰＣにDTL-H2500を接続してください。

   D-2. ボードの初期化

     DTL-H2500は、ファームウェアを最新のバージョンにupdateしてください。
    起動可能なフロッピーディスクに deci\flash25以下をコピーし以下のコマ
    ンドを実行することによって updateできます。
       > cd deci\flash25
       > flash

   D-3. ドライバーのインストール
     deci\psxNTpci のしたの psxNTpci.infを右クリックしてください。
     メニューの中にインストールがあるので選択してください。
     ドライバーのインストールは終了です。


 ◆ ドライバの使用を一時停止する
   SDevTC のツールを使用する場合などはドライバーを切り放してください。
   ドライバを切り替える毎にファームウェアを適切なものに入れ替える必要が
  あります。ファームウェアのバージョンについてはそれぞれのツールの説明を
  お読みください。

  A.B. Ｗｉｎｄｏｗｓ９５ で各種ドライバーの使用を停止する。
       スタートメニュー -> 設定 -> コントロールパネル
       システム -> デバイスマネージャ -> PlayStation
       PlayStation Board DECI (DTL-H2....)をダブルクリックしてください。
       デバイスの使用で"このハードウェア環境で使用不可にする"をチェック
       してください。

  C.D. ＷｉｎｄｏｗｓＮＴ で各種ドライバーの使用を停止する。
       スタートメニュー -> 設定 -> コントロールパネル
       デバイスをダブルクリックしてください。
       PlayStation Board DECI (DTL-H2....)を選択してください。
       状態が開始になっている場合は停止を押してください。
       スタートアップボタンを押してください。手動に切り替えます。


 ◆ ドライバの使用を再開する
   上記手段でドライバを一時停止させている場合には以下の操作で使用可能に
  なります。
   ファームウェアをインストールの手順と同様にして入れ替えておいてください。

  A.B. Ｗｉｎｄｏｗｓ９５ で各種ドライバーの使用を再開する。
       スタートメニュー -> 設定 -> コントロールパネル
       システム -> デバイスマネージャ -> PlayStation
       PlayStation Board deci (dtl-H2....)をダブルクリックしてください。
       デバイスの使用で"このハードウェア環境で使用不可にする"のチェック
       をはずしてください。

  C.D. ＷｉｎｄｏｗｓＮＴ で各種ドライバーの使用を再開する。
       スタートメニュー -> 設定 -> コントロールパネル
       デバイスをダブルクリックしてください。
       PlayStation Board DECI (DTL-H2....)を選択してください。
       スタートアップボタンを押してください。システムに切り替えます。
       開始を押してください。


 ◆ アンインストール

  A.B. Ｗｉｎｄｏｗｓ９５ で各種ドライバーをアンインストールする。
       スタートメニュー -> 設定 -> コントロールパネル
       システム -> デバイスマネージャ -> PlayStation
       PlayStation Board DECI (DTL-H2....)を選択し、削除を押してください。
       Windowsのシステムディレクトリの下に inf\otherというディレクトリが
       存在した場合はそこに設定ファイルが残ってますので削除してください。
       なお Windows95のバージョンによっては infディレクトリの下に oemxx.inf
       という名前に変更された設定ファイルが残ってますので削除してください。

  C.D. ＷｉｎｄｏｗｓＮＴ で各種ドライバーをアンインストールする。
       スタートメニュー -> 設定 -> コントロールパネル
       デバイスをダブルクリックしてください。
       PlayStation Board DECI (DTL-H2....)を選択してください。
       スタートアップボタンを押してください。無効に切り替えます。
