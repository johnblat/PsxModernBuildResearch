このディレクトリには、sgiconv.taz が含まれます。
これは、以下のファイルを tar でまとめ、compress で圧縮したものです。

このアーカイブを展開するには、"sgiconv.taz" を、"sgiconv.tar.Z" にファイル名
変更してから、"zcat sgiconv.tar.Z|tar xvf -" してください。
