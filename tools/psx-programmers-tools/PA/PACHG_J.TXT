 Performance Analyzer Version 2.06 リリースノート


 特徴 

 1) ハイライト機能が追加されました。
    シンボル、波形の名称部分を左クリックすることにより、ハイライトが表示
    されます。右クリックすると、ハイライトは非表示となります。

 2) パケットビューアーの表示に関する不具合を修正しました。


 Performance Analyzer Version 2.05 までの変更点

 1) Driver 対応により Windows 95/98/NT4.0 で動作可能です。

 2) ビデオメモリアナリシス及びGPUパケットアナリシスの表示に関する不具合を
    修正しました。

 3) ビデオメモリビューアーの表示に関する不具合を修正しました。

 4) Windowsのフォント設定による不具合を修正しました。

 5) パケットビューアーに拡大機能を追加しました。


 既知不具合情報

 1) IntelliMouse には対応していません。
