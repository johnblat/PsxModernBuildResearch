サンプルデータ - tuto3.pad について
-----------------------------------------------------------------------

     アナライザを使用して計測されたデータファイルです。
     このデータファイル tuto3.pad は \psx\sample\graphics\texaddr\wave
     の tuto3 を計測したものです。（およそ２Ｖブランク分）

     DTL-H2700 をお持ちでない方でも、アナライザを使用してこのデータ
     ファイルをご覧になれます。
