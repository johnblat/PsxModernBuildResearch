 Performance Analyzer Version 2.06

 ◆ はじめに
   Performance Analyzer Version 2.06 は、DTL-H2700 用のアプリケーションです。
  ドライバー対応により、Windows 95/98/NT で利用できます。


 ◆ ＯＳ，機種
   以下の２つの組合せに分類してそれぞれ説明してありますので、お手元の
  ＯＳ，機種にしたがって最適なものを選択してお読みください。

  A. Windows 95/98 と DTL-H2700 の組合せ
  B. Windows NT と DTL-H2700 の組合せ


 ◆ ドライバーのインストール

  A. Windows 95/98 に DTL-H2700 を接続する。

   A-1. ハードウェアのセットアップ

     開発ボードの取り扱い説明書、もしくはスタートアップマニュアルを
    参考にしてＰＣにDTL-H2700を接続してください。

   A-2. I/O Adress の指定
     pa\psx95pa をハードディスクの適当なところへコピーしてください。
     コピーした pa\psx95pa\psx95pa.infをダブルクリックする。

     IOConfig=で始まっている行を探してください。

     ここに I/O アドレスを指定してください。IOConfig=124c-124f(3ff::)
    などのようにしてください。ディップスイッチで設定した値に 0x0cを加えて
    ください。

     初期値は I/O アドレス:1340となってます。(設定値は 134c)

   A-3. ドライバーのインストール
       スタートメニュー -> 設定 -> コントロールパネル
       ハードウェアをダブルクリックしてください。
       インストールを始めるで "次へ"を選択してください。
       自動的に検出で "いいえ"を選択して "次へ"を選択してください。
       "その他のデバイス"を選択して "次へ"を選択してください。
       "ディスク使用"を選択してください。
       配布ファイルのコピー元で 先程(A-2.)の設定をした psx95pa.infのある
       ディレクトリを指定してください。
       PlayStation Board PA (DTL-H2700)で "次へ"を選択してください。
       ポートアドレスが表示されますので確認して "次へ"を選択してください。
       完了をおして、ドライバーのインストールは終了です。


  B. Windows NT に DTL-H2700 を接続する。

   B-1. ハードウェアのセットアップ

     開発ボードの取り扱い説明書、もしくはスタートアップマニュアルを
    参考にしてＰＣにDTL-H2700を接続してください。

   B-2. I/O Adress の指定
     pa\psxNTpa をハードディスクの適当なところへコピーしてください。
     コピーした pa\psxNTpa\psxNTpa.infをダブルクリックする。

     ISAAddress=で始まっている行を探してください。

     ここに I/O アドレスを指定してください。ISAAdress=0x124c などのように
    してください。ディップスイッチで設定した値に 0x0cを加えてください。

     初期値は I/O アドレス:1340となってます。(設定値は 134c)

   B-3. ドライバーのインストール
     先程(B-2.)の設定をした psxNTpa.infを右クリックしてください。
     メニューの中にインストールがあるので選択してください。
     ドライバーのインストールは終了です。


 ◆ 実行ファイルのインストール
   pa\bin の下にある pa32.exe と libpa.dll を適当なディレクトリへコピーして
  ください。
